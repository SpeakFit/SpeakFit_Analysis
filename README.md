## 🎙️ 음성 데이터 추출 및 분석 로직 (Data Extraction & Analysis Pipeline)

`SpeakFit`은 사용자의 고유한 음색(Baseline)을 분석하고, 데이터 기반의 보정값과 전문가의 전달력 지표를 결합하여 사용자 맞춤형 발표 가이드를 생성합니다. 

본 프로젝트는 발화의 5대 지표인 **WPM(말하기 속도), Pitch(음고), dB(성량), Pause(휴지기 비율), ZCR(발음 선명도)**을 기준으로 데이터를 정제 및 분석합니다.
> ⚠️ **Data Cleansing**: 분석 신뢰도를 높이기 위해 휴지기(Pause)가 비정상적으로 길거나, 전체 발화 시간이 기준치 미만인 노이즈 데이터는 전처리 단계에서 제외하였습니다.

---

### 1. 데이터셋별 분석 및 지표 추출 정의

#### 🏫 대학 강의 데이터 (Style Clustering)
* **추출 지표**: `WPM`, `Pitch`
* **분석 방법**: $K\text{-means}\ (K=4)$ 알고리즘 적용
* **분석 결과**: 대학 강의 음성 분석을 통해 발표 스타일을 아래의 **4가지 군집(Clustering)**으로 정형화하였습니다.
  1. 전달력 있는 스타일
  2. 열정적인 스타일
  3. 신중한 스타일
  4. 지적인 스타일

#### 💬 자유 대화 및 한국인 외래어 발화 데이터 (Offset Calibration)
* **추출 지표**: `WPM`, `Pitch`
* **분석 목적**: 일반적인 일상 발화의 평균 지표를 수립하고, 대학 강의 데이터로 도출된 4가지 발표 스타일 중심점과의 **상대적 차이값(Offset)**을 도출합니다.
* **보정 메커니즘**: 일상 톤에서 특정 발표 스타일로 전환 시 필요한 지표 변화율을 수치화합니다. (예: 열정적인 스타일 진입 시 $\text{WPM} \times 1.2$, $\text{Pitch} + 3\text{Hz}$)
* **차별성**: 추출된 보정값은 모든 사용자에게 획일적으로 적용되지 않으며, 사용자의 고유 `Baseline`에 가변적으로 연산되어 개인 맞춤형 타겟 수치를 형성합니다.

#### 📺 뉴스 앵커 데이터 (Gold Standard Guide)
* **추출 지표**: `dB`, `Pause`, `ZCR`
* **분석 목적**: 전문 스피치 교육을 이수한 앵커 집단의 균일한 발화 특성을 분석하여, 발표 시 지향해야 할 **보편적 전달력의 마스터 지표(Gold Standard)**를 수립합니다.
* **특징**: 앵커 데이터는 발화의 균일성이 높아 스타일별 군집화 대신, 성량의 안정성(`dB`), 문장 간 황금 휴지기 비율(`Pause`), 자·모음 분리도 및 선명도(`ZCR`)의 절대적인 가이드라인 스케일로 활용됩니다.

### 2. 가이드 생성 프로세스 (Pipeline Matrix)

`SpeakFit` 서비스의 가이드 생성 알고리즘은 아래의 매트릭스 구조를 바인딩하여 최종 결과물을 도출합니다.

| 분류 축 | 활용 데이터셋 | 가이드 반영 방식 | 적용 지표 |
| :--- | :--- | :--- | :--- |
| **발표 스타일** <br>(가변 지표) | 대학 강의 데이터 <br> 자유 대화 데이터 | $\text{사용자 Baseline} + \text{스타일별 가변 보정값}$ | `WPM`, `Pitch` |
| **전달력 가이드** <br>(고정 지표) | 뉴스 앵커 데이터 | 전문가 그룹의 상향 평준화된 표준 가이드라인 매핑 | `dB`, `Pause`, `ZCR` |
